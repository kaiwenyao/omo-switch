<objective>
Implement model editing with inline input, fallback_models update, and auto-save.

Purpose: Allow users to edit the model field for selected agents/categories, updating both primary model and all fallback_models, with immediate JSON persistence.
Output: tui/edit.go with edit state machine, updated AppModel, auto-save in main.go.
</objective>

<context>
@config/config.go (from Plan 01)
@tui/app.go (from Plan 02, 03)
@tui/tree.go (from Plan 02)

Edit flow:
1. User presses Enter on agent/category (not provider)
2. Current model value shown as editable text field
3. User types new model OR pastes model string
4. User presses Enter to confirm or Escape to cancel
5. On confirm: update primary model AND all fallback_models to new value
6. Auto-save to JSON file immediately
7. Re-render tree with updated values

Model format: "provider/model-name" (e.g., "ollama-cloud/glm-5.1")
</context>
