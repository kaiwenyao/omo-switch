<objective>
Implement vim-style navigation (h/j/k/l) and batch selection (spacebar).

Purpose: Allow users to navigate the tree efficiently using vim keys and select multiple nodes for batch operations.
Output: tui/keys.go with HandleKey function, updated AppModel with selection state.
</objective>

<context>
@config/config.go (from Plan 01)
@tui/tree.go (from Plan 02)
@tui/app.go (from Plan 02)

Navigation requirements:
- h: move to parent node (if expanded) or previous sibling
- j: move down (next item in FlatList)
- k: move up (previous item in FlatList)
- l: move to child node (if provider, expand/enter)
- Arrow keys: same as j/k (down/up)

Selection requirements:
- Spacebar: toggle selection on current node
- Selected nodes tracked in AppModel
- Selection persists during navigation
</context>
